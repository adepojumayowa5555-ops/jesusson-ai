# 🚀 Jesus'son AI - Professional Assistant

**Jesus'son AI** is a modern, high-performance web-based AI assistant. It is designed to provide a seamless, interactive experience for users needing quick answers, coding help, or image analysis.

## ✨ Key Features
* **Intelligent Conversations:** Powered by the Google Gemini 1.5 Flash API.
* **Multimodal Capabilities:** Upload images and ask the AI to analyze them (Charts, Code, Photos).
* **Persistent Interface:** Full-screen responsive design with a sidebar for chat history.
* **Pro UI:** Clean dark-mode aesthetic inspired by modern developer dashboards.

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **AI Engine:** Google Generative AI (Gemini SDK).
* **Hosting:** GitHub Pages.
